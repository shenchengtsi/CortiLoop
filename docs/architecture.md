## 2. retain() 完整数据流

```
用户/Agent 输入: "Alice 是 ProjectX 的 PM，她偏好用 Figma 做设计"
                    │
                    ▼
    ┏━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┓
    ┃  Step 1: 注意力门控 (Attention Gate) ┃ ← 前额叶注意力筛选
    ┃                                     ┃
    ┃  评分维度:                           ┃
    ┃  · 纠错信号 correction = 0.0         ┃   (无"不对/应该是")
    ┃  · 显式请求 explicit   = 0.0         ┃   (无"记住")
    ┃  · 新奇度   novelty    = 0.62        ┃   (较长,有新实体)
    ┃  · 情感强度 emotion    = 0.0         ┃   (无情感词)
    ┃  · 任务相关 task_rel   = 0.5         ┃   (无任务上下文)
    ┃                                     ┃
    ┃  importance = 0.23 > threshold 0.2   ┃
    ┃  → ✅ 通过门控                       ┃
    ┗━━━━━━━━━━━━━━┳━━━━━━━━━━━━━━━━━━━━━┛
                    │
                    ▼
    ┏━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┓
    ┃  Step 2: 编码 (Encoder)              ┃ ← 海马体编码
    ┃                                     ┃
    ┃  LLM 提取结构化事实:                 ┃
    ┃  ┌─────────────────────────────┐    ┃
    ┃  │ Fact 1:                     │    ┃
    ┃  │ "Alice 是 ProjectX 的 PM"   │    ┃
    ┃  │ entities: [Alice, ProjectX] │    ┃
    ┃  │ source: user_said           │    ┃
    ┃  ├─────────────────────────────┤    ┃
    ┃  │ Fact 2:                     │    ┃
    ┃  │ "Alice 偏好用 Figma 做设计"  │    ┃
    ┃  │ entities: [Alice, Figma]    │    ┃
    ┃  │ source: user_said           │    ┃
    ┃  └─────────────────────────────┘    ┃
    ┃                                     ┃
    ┃  批量生成 Embedding (text-embed-3)   ┃
    ┃  附加 EncodingContext:               ┃
    ┃  · session_id, task, entities        ┃
    ┗━━━━━━━━━━━━━━┳━━━━━━━━━━━━━━━━━━━━━┛
                    │
                    ▼

    ┃                                     ┃
    ┃  memory_units 表:                    ┃
    ┃  ┌─────────────────────────────────┐┃
    ┃  │ id: uuid-001                    │┃
    ┃  │ content: "Alice 是 ProjectX..." │┃
    ┃  │ importance: 0.23                │┃

    ┃  │ embedding: [0.012, -0.034, ...] │┃
    ┃  │ base_strength: 1.0              │┃
    ┃  │ decay_rate: 0.1 (episodic)      │┃
    ┃  │ state: ACTIVE                   │┃
    ┃  └─────────────────────────────────┘┃
    ┃  (同样写入 uuid-002)                 ┃
    ┃                                     ┃
    ┃  → 写入后立即可检索 ✅               ┃

    ┃  Step 4: 关联构建 (Association)      ┃ ← 赫布学习
    ┃                                     ┃
    ┃  共现边 (co_occurrence):             ┃
    ┃  Alice ──── ProjectX  weight=1.0     ┃
    ┃  Alice ──── Figma     weight=1.0     ┃

                    ▼

    ┃                                     ┃
    ┃  LLM 判断:                           ┃
    ┃  ┌─────────────────────────────┐    ┃
    ┃  │ CREATE Observation:         │    ┃

    ┃  │ source_units: [001, 002]    │    ┃
    ┃  │ decay_rate: 0.03 (semantic) │    ┃
    ┃  └─────────────────────────────┘    ┃
    ┃                                     ┃
    ┃  observations 表写入                 ┃
    ┗━━━━━━━━━━━━━━┳━━━━━━━━━━━━━━━━━━━━━┛
                    │
                    ▼
    ┏━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┓
    ┃  Step 6: 再巩固检查 (Reconsolidation)┃ ← 记忆更新窗口
    ┃                                     ┃

    ┃                                     ┃

    ┃  │ → 旧 Observation 保留到      │    ┃

    ┃  │ → 内容更新为新信息           │    ┃
    ┃  │ → conflict 记录写入          │    ┃
    ┃  └─────────────────────────────┘    ┃
    ┃  (如无冲突则跳过此步)                ┃
    ┗━━━━━━━━━━━━━━┳━━━━━━━━━━━━━━━━━━━━━┛
                    │
                    ▼

              }
```

```
查询: "Alice 最近在做什么项目？"
           │
           ▼
┏━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┓
┃                    查询预处理                                ┃
┃  · 生成 query embedding                                    ┃
┃  · 提取关键词: ["Alice", "最近", "项目"]                     ┃
┃  · 检测时间意图: "最近" → last 7 days                       ┃
┗━━━━━━━━━━━━━━━━━━━━━┳━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛
                       │
         ┌─────────────┼─────────────┬────────────┐
         ▼             ▼             ▼            ▼
┌──────────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐
│  Route 1     │ │ Route 2  │ │ Route 3  │ │ Route 4  │
│  语义检索    │ │ 关键词   │ │ 图谱遍历 │ │ 时序过滤 │
│  (Semantic)  │ │ (BM25)   │ │ (Graph)  │ │(Temporal) │
│              │ │          │ │          │ │          │
│ query embed  │ │ LIKE     │ │扩散激活   │ │ 7天内的  │
│ vs all units │ │ "%Alice%"│ │从语义top5 │ │ 记忆     │
│ cosine sim   │ │ "%项目%" │ │出发，沿边 │ │          │
│              │ │          │ │跳2层     │ │          │
│ weight: 0.4  │ │ wt: 0.2 │ │ wt: 0.25│ │ wt: 0.15│
│              │ │          │ │          │ │          │
│ ┌──────────┐ │ │┌────────┐│ │┌────────┐│ │┌────────┐│
│ │unit:0.92 │ │ ││unit:1.0││ ││obs:0.5 ││ ││unit:1.0││
│ │obs:0.88  │ │ ││unit:1.0││ ││unit:0.3││ ││unit:1.0││
│ │obs:0.85  │ │ │└────────┘│ │└────────┘│ │└────────┘│
│ │unit:0.71 │ │ │          │ │          │ │          │
│ └──────────┘ │ │          │ │          │ │          │
└──────┬───────┘ └────┬─────┘ └────┬─────┘ └────┬─────┘
       │              │            │             │
       └──────────────┴──────┬─────┴─────────────┘
                             │
                             ▼
              ┏━━━━━━━━━━━━━━━━━━━━━━━━━━━━┓
              ┃  Reciprocal Rank Fusion     ┃
              ┃                            ┃
              ┃  score(d) = Σ wᵢ/(k+rankᵢ) ┃

              ┃                            ┃
              ┃  ┌────────────────────────┐ ┃
```

```
┃                     ┃                │

┃                     ┃                │

┃ │ strength ≤ 0.1  │→│ COLD           │
┃ └─────────────────┘ ┃                │
┃                     ┃                │
┃ 不同衰减速率:        ┃                │
┃ · episodic  λ=0.1  ┃  (7天半衰期)    │
┃ · semantic  λ=0.03 ┃  (23天半衰期)   │
┃ · procedural λ=0.005┃ (139天半衰期)  │

┃ (Pruning)           ┃                │
┃                     ┃                │
┃ ┌─────────────────┐ ┃                │
┃ │ 去重             │ ┃                │
┃ │ embedding 余弦   │ ┃                │
┃ │ > 0.92 的pair    │ ┃                │
┃ │ → 保留访问多的   │ ┃                │
┃ │ → archive 另一个 │ ┃                │
```

```
strength
  1.0 ┤●─── 首次存储
      │ \
```
